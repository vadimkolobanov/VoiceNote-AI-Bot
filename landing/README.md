# Лендинг — перенесён

Этот каталог (`landing/index.html` + `tokens.css`) был временным черновиком,
сделанным через Claude Design до того, как мы интегрировали продуктовую
страницу в общий лендинг Метод­екса.

## Где живёт сейчас

Страница «Methodex Секретарь» теперь часть монорепо `metodex-frontend`
(Next.js 16 + Tailwind 4):

- Маршрут: `/secretary`
- Компоненты: `src/components/secretary/`
  - `Hero.tsx` — заголовок + мокап телефона с анимированным feed
  - `Features.tsx` — 9 плиток (голос/память/привычки/просрочка/спроси-меня/inline-кнопки/расскажи-о-себе/удалить)
  - `AppPreview.tsx` — fake-окно «Что я о тебе знаю» с фактами/сериями привычек
  - `Pricing.tsx` — Free + Pro 390₽/мес
  - `FAQ.tsx` — 9 вопросов
  - `CTA.tsx` — финальный download-блок
- Карточка продукта добавлена в `src/lib/products.ts` (slug: `secretary`,
  category `ai`)
